# GLSL-Particle-System

[ Download all files in this section. ]
It includes the GLSL-Particle-System Program, the engine needed to run it, and a shortcut to the executable.

Controls:

W: Move forward
S: Move backward
A: Strafe left.
D: Strafe right
E: Move up
Space: Move down
Shift: Movement speed boost
Mouse: Look around

~: Show console and log
P: Pause/unpause
R: Toggle rendering
F: Show/Hide FPS
X: Enable/disable polygon_lines
C: Lock/Unlock mouse from window.
ESC: Exit application.
